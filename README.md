# Windor UMKM Application

Aplikasi listing direktori UMKM yang berbasis lingkungan sekitar Universitas Islam Negri Syarif Hidayatullah Jakarta.  
Windor adalah sebuah website listing UMKM yang memudahkan mahasiswa di daerah UIN Jakarta untuk memesan makanan dari vendor yang jauh dengan mudah. Windor menyediakan fitur pemesanan online serta metode pembayaran tersendiri yang dilengkapi poin rewards yang menjadi insentif bagi pengguna untuk terus menggunakan platform kita.

Submisi untuk kompetisi Web In Action 2025.

Repository ini memuat source code untuk bagian frontend dari aplikasi yang dibuat menggunakan framework React.js
Untuk backend API dari aplikasi bisa diakses pada repositori [windor-api](https://github.com/greyseth/windor-api).

# Petunjuk Instalasi
### 1. **Clone Repository**
```bash
git clone https://github.com/greyseth/windor.git
```
### 2. **Masuk Direktori Project**
```bash
cd windor
```
### 3. **Instal Dependensi**
Pastikan Node.js dan NPM atau yarn sudah terinstal pada sistem Anda.
```bash
npm install
```
ATAU
```bash
yarn install
```
### 4. **Jalankan Project**
```bash
npm run dev
```
ATAU
```bash
yarn start
```
Maka aplikasi akan berjalan dan dapat diakses pada ``http://localhost:5173``
